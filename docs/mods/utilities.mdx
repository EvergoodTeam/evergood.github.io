## üìë Description {#description}

Evergood Utilities is a simple Minecraft Mod for 1.12.2 consisting of a collection of Items, Blocks, Fluids and recipes created as integration between other Mods, primarly used by the Evergood Modpacks.

We also provide some commands to speed up some repetitive tasks.

<p><Image img={require('@site/static/img/utilitiesBanner2.png')} /></p>


**Currently supported mods are**:

- [Tinkers' Construst](https://www.curseforge.com/minecraft/mc-mods/tinkers-construct)
- [Thermal Expansion](https://www.curseforge.com/minecraft/mc-mods/thermal-expansion) / [Foundation](https://www.curseforge.com/minecraft/mc-mods/thermal-foundation)
- [Astral Sorcery](https://www.curseforge.com/minecraft/mc-mods/astral-sorcery)
- [Mekanism](https://www.curseforge.com/minecraft/mc-mods/mekanism)
- [Immersive Engineering](https://www.curseforge.com/minecraft/mc-mods/immersive-engineering)
- [Nuclearcraft](https://www.curseforge.com/minecraft/mc-mods/nuclearcraft-mod)
- [Biomes O' Plenty](https://www.curseforge.com/minecraft/mc-mods/biomes-o-plenty)
- [Extra Planets](https://www.curseforge.com/minecraft/mc-mods/extraplanets)

**Current additions**:

- Zinc, Palladium, Mercury, Titanium
- Cinnabar
- Starmetal / Aquamarine Blocks
- Nuggets for Nuclearcraft Ingots, BOP Gems and for Lapis
- Items used as Icons (useful for a Quest Book)
- Commands to make some operations faster (making it always daytime, nightvision, spectator)
