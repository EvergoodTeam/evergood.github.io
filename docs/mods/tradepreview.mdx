## ğŸ“‘ Description {#description}

Check all the trades a villager has to offer from the press of a key without having to open their inventory, all of this with a highly configurable overlay!

- You can move around the overlay by pressing the assigned key (<code>M</code> by default)
- You can scroll through the offered enchanted items with the assigned keys (<code>UP</code> and <code>DOWN</code> arrows by default) or use the scroll wheel!
- Enchanted books have their names displayed automatically, removing the need to scroll through them
- You can change the colours used (for the container and enchanted item selection) from the configs menu
- All of the keys can be configured in the options menu

Currently only available for the **Fabric ModLoader**.
