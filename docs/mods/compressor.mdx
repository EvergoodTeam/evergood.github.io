## üìë Description {#description}

Compressor is a Minecraft Mod that introduces compressed blocks (up to 8x compressed variants!) for commonly found resources.
This allows the user to store millions of blocks (**43,046,721** blocks for an octuple compressed block) in a single one!

Out of the box you will be provided with **recipes** to craft and uncraft compressed blocks, as well as **global tags** for each entry to create your very own custom recipes.

Currently Compressor is only available for the **Fabric ModLoader** and requires the [**Chassis**](./chassis.mdx) library.

:::tip

If you want more blocks to be supported let us know!

:::
